# Rift Deck

Rift Deck is a Vite + React app.

## Local Development

1. Clone the repository.
2. Install dependencies:

```bash
npm install
```

3. Create an `.env.local` file with the required Supabase and app environment variables.
4. Start the development server:

```bash
npm run dev
```

## Build

```bash
npm run build
```
